技术实现	设计思路
- 4bit NF4量化 + 双重量化	降低75%显存占用，消费级显卡可部署
- 自动设备映射（device_map="auto"）	自适应CPU/GPU环境，提高兼容性
- float16计算精度保留	平衡量化误差与计算效率
